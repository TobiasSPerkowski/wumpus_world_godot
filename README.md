## Descrição:
Implementação do Mundo Wumpus feita na Godot, com sistema de ajuda ao jogador via script em Python usando pysat. 
Baseado no problema descrito em https://moj.naquadah.com.br/new/treino/problem/?id=flia-problems.wumpus-medio

## Controles:
 - A/D: girar para esquerda/direita
 - W: mover para frente
 - Espaço: atirar flecha
 - H: pedir ajuda
